## Summary

* 이 프로젝트는 판교에서 진행하는 북스터디모임 공부내용의 기록이다.
* 작성자가 읽는 책은 [실전 아파치 카프카][book].
* 총 5명이 참가하며 읽어온 분량 만큼 서로에게 설명하고 토론한다.

## What Code?

* [실전 아파치 카프카][book] 의 예제다.
* 챕터 별로 나눠서 예제를 진행한다.
* 챕터에 따라 예제가 없는건 패키지만 있을 수 있다.
* 릴리즈는 챕터 기준으로 한다.
